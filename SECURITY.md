# Security Policy

## Supported Versions

The project maintains the latest release line. Security fixes are provided for
the most recent release only.

## Reporting a Vulnerability

If you discover a security issue, please report it responsibly:

1. Open a GitHub Security Advisory for this repository (preferred), or
2. Open a GitHub issue if you are unable to create a private report.

Please include as much detail as possible so we can reproduce and address the
issue quickly.
